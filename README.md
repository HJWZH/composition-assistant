## ✨ 项目亮点
- **深度语义理解**：突破关键词匹配局限，理解"挫折→逆境成长"的抽象关联
- **动态学习系统**：10轮迭代训练机制，持续提升素材推荐精准度
- **多维度过滤**：类别/主题/相似度三级检索体系
- **轻量化部署**：预计算嵌入向量技术，CPU环境0.5秒响应
